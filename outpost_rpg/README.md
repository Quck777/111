# 🎮 РПГ Игра "ФОРПОСТ"

Пиксельная браузерная многопользовательская РПГ игра с элементами экономики, боя, исследования мира и социального взаимодействия.

## 📋 Технологии

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** PHP 8+
- **База данных:** MySQL 8+
- **Стиль:** Пиксель-арт дизайн

## ✨ Возможности

### Реализовано на данном этапе:

1. **База данных**
   - Полная схема БД со всеми таблицами
   - Таблицы: пользователи, предметы, инвентарь, локации, монстры, чат, квесты, достижения, рынок, гильдии, PvP бои и др.
   - Триггеры и представления
   - Лидерборд

2. **Аутентификация**
   - Регистрация новых пользователей
   - Вход/выход
   - Сессии
   - Безопасность паролей (bcrypt)

3. **Пользовательская система**
   - Характеристики (сила, ловкость, интеллект, выносливость, удача)
   - Опыт и уровни
   - Ранги с бонусами
   - Энергия
   - Золото и драгоценные камни

4. **Дизайн**
   - Пиксель-арт стиль
   - Адаптивная верстка
   - Анимации
   - Темная тема

5. **Клиентская часть**
   - Утилиты JavaScript
   - API клиент для запросов
   - Основной игровой цикл
   - Чат (реальное время)
   - Инвентарь с слотами
   - Панель персонажа

## 🚀 Установка

### Требования

- PHP 8.0 или выше
- MySQL 8.0 или выше
- Веб-сервер (Apache/Nginx)
- Composer (опционально)

### Шаг 1: Клонирование репозитория

```bash
cd /workspace
# Проект уже создан в outpost_rpg/
```

### Шаг 2: Настройка базы данных

1. Создайте базу данных:

```sql
CREATE DATABASE outpost_rpg CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. Импортируйте схему:

```bash
mysql -u root -p outpost_rpg < database/schema.sql
```

Или выполните SQL файл через phpMyAdmin.

### Шаг 3: Настройка конфигурации

Откройте файл `includes/config.php` и настройте параметры подключения к БД:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'outpost_rpg');
define('DB_USER', 'ваш_пользователь');
define('DB_PASS', 'ваш_пароль');
define('DB_CHARSET', 'utf8mb4');
```

Также измените `SITE_URL` на ваш домен:

```php
define('SITE_URL', 'http://yourdomain.com/outpost_rpg');
```

### Шаг 4: Права доступа

Убедитесь, что веб-сервер имеет права на запись в директорию проекта:

```bash
chmod -R 755 /workspace/outpost_rpg
chown -R www-data:www-data /workspace/outpost_rpg
```

### Шаг 5: Запуск

Откройте в браузере:

```
http://localhost/outpost_rpg
```
