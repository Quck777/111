## 🚀 Запуск проекта

### Требования
- PHP 8.0+
- MySQL 5.7+ / MariaDB 10.3+
- Веб-сервер (Apache/Nginx)

### Установка

1. **Импорт базы данных:**
```bash
mysql -u root -p outpost_rpg < database/schema.sql
```

2. **Настройка конфигурации:**
Отредактируйте `includes/config.php`:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'outpost_rpg');
define('DB_USER', 'ваш_пользователь');
define('DB_PASS', 'ваш_пароль');
define('SITE_URL', 'http://your-domain.com');
```

3. **Права доступа:**
```bash
chmod 755 /path/to/outpost_rpg
chmod 644 includes/config.php
```

4. **Откройте в браузере:**
```
http://localhost/outpost_rpg
```

### Авторизация
- `POST /api/auth/login.php` - Вход
